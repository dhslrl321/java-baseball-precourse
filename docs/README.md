# 기능 요구사항

- 1~9 까지 서로 다른 수로 이루어진 3자리의 수를 맞춘다
  - 스트라이크 : 같은 수가 같은 자리
  - 볼 : 같은 수가 다른 자리
  - 낫싱 : 같은 수가 전혀 없을 경우
- 컴퓨터의 수를 모두 맞추면 승리한다
- 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException` 을 발생시키고 종료되어야 한다.
